# 그릿 프로젝트 체크리스트

---

## 준비기간 (~ 5/3 자정까지)

- [ ] 목적설명서 1차 완성 (`목적설명서_1차.md`)
- [ ] 아이덴티티 네임 확정 (후보: Scout / Navigator / Bridge / 길잡이)
- [ ] 카톡방에 목적설명서 + 아이덴티티 네임 공유
- [ ] 팀원들 트리거 읽기
- [ ] 트리거 1~10 전체 복기
- [ ] 앞으로 관심 가질 산업/기업 후보 3개 이상 리서치

황성철 WHW 기반 관심 산업 후보:
- 외국인 대상 생활 서비스 / 한국 정착 지원
- 공간 경험 기획 / 로컬 브랜딩
- 리서치 & 연결 플랫폼 / 커뮤니티
- 문화 교류 / 이민자 지원

---

## 1회차 (5/2 토요일 시작)

- [ ] 사전과제 + 워크숍 시트 + 트리거 + 1차 목적설명서 공유
- [ ] 팀원 피드백 받기 → WHW업데이트 폴더에 기록
- [ ] 앞으로 관심 가지고 볼 산업/기업/트렌드 공유
- [ ] 다음 주 함께 참여해볼 세미나&강의 리스트업

---

## 2회차

- [ ] 산업/기업/트렌드 조사 1건 이상 완성 → `기업조사/` 폴더에 저장
- [ ] 트렌드 조사 1건 이상 → `산업트렌드/` 폴더에 저장
- [ ] 조사 내용 팀원과 공유
- [ ] 서로의 WHW에 5why 피드백
- [ ] WHW 현재버전 업데이트 (`WHW업데이트/WHW_현재버전.md`)
- [ ] 다음 주 세미나&강의 리스트업

---

## 3회차

- [ ] 산업/기업/트렌드 조사 추가
- [ ] 필독서 읽고 공유 또는 공간 탐방
- [ ] WHW 현재버전 업데이트
- [ ] 다음 주 세미나&강의 리스트업

---

## 4회차 (최종)

- [ ] 최종 아이덴티티 휠 정리
- [ ] 목적설명서 최종본 완성
- [ ] 액션플랜 수립
- [ ] 한달 팀플 정리 & 발표 준비
